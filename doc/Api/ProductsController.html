Products

Returns the stock available for a product.

/api/v1/products/get_stock [POST]

Required Parameters

  • api_key String Agent's authentication token. This token allows us to check if the agent is allowed to request this action.
  • sku String Sku of the product we are interested in. This parameter is the product description.
  • club_id String Id of the club the product belongs to.

Response Fields

  • stock Integer Actual stock of the product. This value is an integer type. This will returned only when the method run without finding any kind of error.
  • allow_backorder Integer Flag to inform that product allow negative stocks. It returns 1 for true value, and 0 for false value. This flag will returned only when there was no error.
  • code String Code related to the method result.
  • message String Shows the method errors.

Examples

Example of valid request. Request curl -v -k -X POST -d "api_key=zmemqz1Yi6v6aEm5fLjt&club_id=2&sku=KIT-CARD" https://dev.stoneacrehq.com:3000/api/v1/products/get_stock Response {"code":"000","stock":9746,"allow_backorder":true} Example response to valid request.

Returns the stock available and the backorder flag for a list of product.

  • sku Product's sku.
  • stock Actual stock of the product. This value is an integer type
  • allow_backorder Flag to inform that product allow negative stocks. It returns 1 for true value, and 0 for false value.

/api/v1/products/get_list_of_stock [POST]

Required Parameters

  • api_key String Agent's authentication token. This token allows us to check if the agent is allowed to request this action.
  • sku String product's skus that we are interest in. Skus must be separated by commas. (Eg: "KIT-CARD,NCARFLAGKASEYKAHNE")
  • club_id String Id of the club the products belongs to.

Response Fields

  • product_list Array Array of Hashes with the product's information. This Array is returned if there were no error.
  • skus_could_not_found Array Array with the skus that we were not able to find.
  • code String Code related to the method result.
  • message String Shows the method errors.

Examples

Example of valid request. Request curl -v -k -X POST -d "api_key=zmemqz1Yi6v6aEm5fLjt&club_id=2&sku=KIT-CARD,AnoterOne,carflag" https://dev.stoneacrehq.com:3000/api/v1/products/get_list_of_stock Response {"code":"000","product_list":[{"sku":"KIT-CARD","stock":9746,"allow_backorder":true}],"skus_could_not_found":["AnoterOne","carflag"]} Example response to valid request.
